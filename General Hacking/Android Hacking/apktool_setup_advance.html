Advanced Android Hacking

PART-1 : Installation of apktool

Hey what's going on everybody welcome back to VulnX. This is the first part of the Advanced Android Hacking series. If you have not already seen the basic android hacking series, then go ahead and check it out first as this entire series requires you to have the knowledge of the basic android hacking concepts.

So before we dive into decompiling, changing smali, recompiling, exploiting and all that interesting concepts, we have decided to make the first tutorial on the installation of apktool. Well yes I understand that this seems quite boring, but a lot of people including myself have had issues with it.

We do recommend you to follow this tutorial without being over confident as you might end up getting a lot frustrated when you have completed all the difficult levels of the series and finally when you are waiting for the exploit, at that time the exploit fails.

STEP 1 : Downloading essential files

First of all run the following command in your terminal :- 
wget https://raw.githubusercontent.com/iBotPeaches/Apktool/master/scripts/linux/apktool


and then find & download the latest version of apktool from here, and name it as apktool.jar [ make sure to place it in the same directory as 'apktool' ].

STEP 2 : Shifting files and checking

This step is fairely simple, just type the below command to shift apktool to the bin directory so that you can access it directly via command line by typing 'apktool' :- 
chmod +x apktoo* && mv apktool* /usr/local/bin/


After that simply type 'apktool', if everything goes well you must get an output similar to this :-
